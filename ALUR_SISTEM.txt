📋 ALUR KESELURUHAN SISTEM VALTUS
================================================================================

🎯 OVERVIEW:
   Sistem Valtus mengelola proses pembelian Robux dan item game dari awal 
   pembelian customer sampai item masuk ke akun game mereka. Admin memiliki 
   kontrol penuh untuk memverifikasi pembayaran dan memproses pesanan.

================================================================================
📱 ALUR CUSTOMER (USER SIDE)
================================================================================

🔸 STEP 1: PEMBELIAN ROBUX
   ┌─────────────────────────────────────────────────────────────┐
   │ 1. Customer buka website Valtus                            │
   │ 2. Klik "Beli Robux" atau "Beli Robux Sekarang"           │
   │ 3. Pilih jumlah Robux (Quick Pick atau Custom)            │
   │ 4. Masukkan username Roblox dan klik "Cek"                │
   │ 5. Sistem validasi username via Roblox API                │
   │ 6. Klik "Lanjutkan Pembelian"                             │
   └─────────────────────────────────────────────────────────────┘
   ↓
   ┌─────────────────────────────────────────────────────────────┐
   │ 7. Sistem cek GamePass customer                            │
   │    - Jika GamePass ditemukan → lanjut ke pembayaran       │
   │    - Jika tidak ditemukan → tampilkan modal instruksi     │
   │ 8. Customer buat/atur GamePass sesuai harga yang ditentukan│
   │ 9. Klik "Sudah Mengatur Gamepass"                         │
   └─────────────────────────────────────────────────────────────┘
   ↓
   ┌─────────────────────────────────────────────────────────────┐
   │ 10. Masukkan email untuk notifikasi                       │
   │ 11. Klik "Lanjutkan ke Pembayaran"                        │
   │ 12. Pilih metode pembayaran (Manual/Gateway)              │
   │ 13. Lakukan pembayaran sesuai instruksi                   │
   │ 14. Upload bukti transfer (jika manual)                   │
   │ 15. Klik "Kirim Bukti Transfer"                           │
   └─────────────────────────────────────────────────────────────┘
   ↓
   ┌─────────────────────────────────────────────────────────────┐
   │ 16. Order dibuat dengan status:                            │
   │     - payment_status: "waiting_confirmation"              │
   │     - order_status: null                                  │
   │ 17. Customer mendapat Order ID                            │
   │ 18. Customer bisa cek status di "Cek Pesanan"             │
   └─────────────────────────────────────────────────────────────┘

🔸 STEP 2: PEMBELIAN ITEM GAME LAINNYA
   ┌─────────────────────────────────────────────────────────────┐
   │ 1. Customer pilih produk di homepage                       │
   │ 2. Klik produk yang diinginkan                            │
   │ 3. Masukkan username game, jumlah, dan email              │
   │ 4. Pilih metode pembayaran                                │
   │ 5. Lakukan pembayaran dan upload bukti                    │
   │ 6. Order dibuat dengan status sama seperti Robux          │
   └─────────────────────────────────────────────────────────────┘

================================================================================
🔧 ALUR ADMIN (ADMIN SIDE)
================================================================================

🔸 STEP 1: KONFIRMASI PEMBAYARAN
   ┌─────────────────────────────────────────────────────────────┐
   │ 1. Admin login ke panel admin                             │
   │ 2. Buka menu "Payments"                                   │
   │ 3. Lihat orders dengan status "waiting_confirmation"      │
   │ 4. Klik Order ID untuk melihat detail                     │
   │ 5. Periksa bukti transfer yang diupload customer          │
   │ 6. Verifikasi nominal pembayaran                          │
   └─────────────────────────────────────────────────────────────┘
   ↓
   ┌─────────────────────────────────────────────────────────────┐
   │ 7. Admin pilih aksi:                                      │
   │    A. APPROVE (Setujui):                                  │
   │       - payment_status: "completed"                       │
   │       - order_status: "pending"                           │
   │       - Order masuk ke menu "Orders"                      │
   │    B. REJECT (Tolak):                                     │
   │       - payment_status: "failed"                          │
   │       - order_status: null                                │
   │       - Order dihapus dari sistem                         │
   └─────────────────────────────────────────────────────────────┘

🔸 STEP 2: PROSES ORDER
   ┌─────────────────────────────────────────────────────────────┐
   │ 1. Admin buka menu "Orders"                               │
   │ 2. Lihat orders dengan status "pending"                   │
   │ 3. Klik Order ID untuk melihat detail                     │
   │ 4. Klik tombol "Proses Orderan"                           │
   │ 5. Konfirmasi dengan modal popup                          │
   └─────────────────────────────────────────────────────────────┘
   ↓
   ┌─────────────────────────────────────────────────────────────┐
   │ 6. Sistem otomatis update:                                │
   │    - order_status: "pending" (tetap)                      │
   │    - completed_at: dihitung otomatis                      │
   │    - Robux: berdasarkan setting auto_complete_days        │
   │    - Item lain: 15 jam dari sekarang                      │
   │ 7. Customer mendapat notifikasi:                          │
   │    "Robux akan masuk dalam X hari"                        │
   │    "Item akan masuk dalam X jam"                          │
   └─────────────────────────────────────────────────────────────┘

🔸 STEP 3: PENGIRIMAN ROBUX/ITEM
   ┌─────────────────────────────────────────────────────────────┐
   │ 1. Admin kirim Robux ke akun customer via GamePass        │
   │ 2. Admin kirim item ke akun game customer                 │
   │ 3. Update order_status: "completed"                       │
   │ 4. Customer mendapat notifikasi email                     │
   │ 5. Customer bisa cek status di "Cek Pesanan"              │
   └─────────────────────────────────────────────────────────────┘

================================================================================
⏰ TIMELINE & DURASI
================================================================================

🔸 PEMBAYARAN:
   - Customer upload bukti: Langsung
   - Admin verifikasi: 3-5 jam (manual)
   - Gateway payment: Otomatis

🔸 PROSES ORDER:
   - Admin proses order: Langsung setelah konfirmasi
   - Notifikasi ke customer: Otomatis

🔸 PENGIRIMAN:
   - Robux: 1-5 hari (berdasarkan setting)
   - Item game lain: 15 jam
   - Admin kirim: Manual sesuai timeline
